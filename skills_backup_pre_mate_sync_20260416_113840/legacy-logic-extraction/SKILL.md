# Skill: legacy-logic-extraction

## Purpose
Extract and summarize key legacy Java/JSP logic relevant to module behavior preservation.

## Stage
`findings`

## Extraction Rules
- Focus on business rules, validations, branching, and side effects.
- Link findings to discovered Java artifacts and workflows.
- Highlight high-risk logic that must remain behaviorally equivalent.

## Dependencies
- `java-logic-understanding`

## Outputs
- `legacy-logic-summary.json`
- `legacy-logic-risks.json`
- `result.json`
